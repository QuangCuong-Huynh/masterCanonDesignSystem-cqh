Preamble

Design Philosophy

"Clarity is not emptiness. Softness is not weakness. Every surface is a statement."

This Canon is the single source of truth governing all visual output attributed to Cường.Q.Huỳnh — spanning web UI, academic documents, slide decks, data visualization, and creative works. It is a personal design language and a living IP asset.

Three Governing Principles

#PrincipleStatement
01Soft Minimalism as DefaultRestrained color, generous whitespace, refined typography. Complexity lives in structure, not decoration.
02Contextual MorphismSurface style adapts to medium but palette and typography remain canonical across all contexts.
0360 / 30 / 10 Color LawNon-negotiable. Primary dominates, secondary structures, accent punctuates. No exceptions.
Part I

Color System

1.1 The 60/30/10 Law

Primary (60%) → Base canvas · Secondary (30%) → Structure · Accent (10%) → Signal. Non-negotiable across all themes and contexts.

1.2 Theme Switcher — Click to Apply

Click any theme card to apply it live across this document. The active theme is highlighted. All tokens, surfaces, and typography update instantly.

1.3 Active Theme Tokens

Primary — 60%
--bg · Canvas
Secondary — 30%
--surface · Structure
Accent — 10%
--accent · Signal
Exceptional <1%
#E07A5F · Terracotta

1.4 Exceptional Color Protocol

In FOCUS/CLARITY/NIGHTFALL: #E07A5F (terracotta) reserved for <1% of composition — alert banners, critical callouts, personal signature marks. In WARMTH: primary accent. Never as background or body text.

Violations to Avoid

ViolationStatus
Accent used for large background blocksFORBIDDEN
Secondary used as heading text colorFORBIDDEN
Multiple competing accent colors in one compositionFORBIDDEN
Accent exceeding 15% of visible areaFLAG FOR REVIEW

Part X

Quality Checklist

Color & Theme

  • 60/30/10 law observed across composition
  • Single accent color per composition
  • Exceptional terracotta <1% (or primary accent in WARMTH only)
  • WCAG 2.1 AA contrast verified for all text/background pairs

Typography

  • Maximum 3 typefaces per composition
  • Scale hierarchy: body < subhead < heading
  • Serif for academic body; sans for UI/slides; mono for code

Layout

  • 4px spacing modular scale observed
  • Minimum 16px padding on all major blocks
  • Grid-based structure throughout

Surface & Morphism

  • Correct morphism style for context (Soft-Morphism default)
  • No overly aggressive shadows or contrast
  • Transition durations within spec (0.2–0.4s)

Responsive & Accessible

  • Layout tested at all four breakpoints
  • 44px touch targets confirmed
  • ARIA labels on all non-text interactive elements
  • prefers-reduced-motion respected
  • Keyboard navigation functional

Documents & Traceability

  • Author attribution block present
  • Version and date recorded
  • Traceability footnote included
  • Changelog updated
